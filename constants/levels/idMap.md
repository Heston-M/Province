# Game ID Map

## 0 to 199: Campaign Levels
Ids from 0 to 199 are reserved for campaign levels.

## 800 to 999: Random Levels
Ids from 800 to 999 are reserved for random game configurations.

## > 999: Custom Games
Ids 1000 and above are used for custom games saved by the user. Within this range, the ids in use are managed by the UserContext (see UserContext.tsx).
